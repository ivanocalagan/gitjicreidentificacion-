🔧 CONFIGURACIÓN TÉCNICA
─────────────────────────────────────────────────────────────────────────────

PostCSS (postcss.config.js)
───────────────────────────
Archivo creado para compilar Tailwind CSS:
- Plugin tailwindcss: Compila clases Tailwind
- Plugin autoprefixer: Añade prefijos de navegadores

Tailwind Config (tailwind.config.js)
────────────────────────────────────
- Content paths: ['./index.html', './src/**/*.{js,jsx}']
- Scans todos los archivos jsx para clases
- Genera CSS optimizado solo con clases usadas

Custom CSS (index.css)
─────────────────────
- @tailwind directives: Importa base, components, utilities
- Custom animations: Define float, glow, slideDown, fadeIn
- Custom utilities: .btn-primary, .card, .hover-lift, etc.
- Custom colors y gradientes

📊 CÓMO PROBAR EL FRONTEND
─────────────────────────────────────────────────────────────────────────────

1. INICIAR BACKEND
   cd C:\Users\Ivan\saga\backend
   .\venv\Scripts\python.exe main.py
   → Escucha en http://localhost:8000

2. INICIAR FRONTEND
   cd C:\Users\Ivan\saga\frontend
   npm run dev
   → Se abre en http://localhost:5173 (o 5174 si está ocupado)

3. PROBAR FLUJO COMPLETO
   - Paso 1: Arrastra coopefisc_socios.csv
   - Paso 2: Click en "Evaluar Riesgo"
   - Paso 3: Ajusta epsilon, genera sintéticos
   - Paso 4: Descarga CSV/SQL/PDF
